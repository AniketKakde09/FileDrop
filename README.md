# 🚀 FileDrop

Transfer files instantly across your local network — no cloud, no cables, no accounts.

FileDrop is a lightweight, self-hosted file transfer tool that lets you send files from any browser to your computer over your local Wi-Fi network.

## ✨ Features

* 📱 QR code connection
* 🌐 Works in any modern browser
* 🍎 iPhone & 🤖 Android compatible
* 🐧 Linux & 🪟 Windows support
* 📂 Drag & Drop uploads
* ⚡ Fast local network transfers
* 🔒 No internet required
* 📦 Single executable

## 📥 Download

Download the latest binary from the **Releases** page.

* **Linux (x64):** `filedrop-linux-amd64`
* **Windows (x64):** `filedrop-windows-amd64.exe`

> `https://github.com/AniketKakde09/FileDrop/releases/tag/v.1.0.0`

## 🚀 Quick Start

### Linux

```bash
chmod +x filedrop-linux-amd64
./filedrop-linux-amd64
```

### Windows

Run:

```text
filedrop-windows-amd64.exe
```

Then scan the QR code shown in the terminal (or open the displayed URL) and start transferring files.

Uploaded files are saved in the `uploads/` directory.
